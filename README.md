# 📚 نظام إدارة متكامل سنتر الطالب

## الميزات الرئيسية ✨

### 1️⃣ نظام الحضور والغياب
- 📱 قراءة الباركود التلقائية
- ✏️ تسجيل يدوي للحضور والغياب
- 📷 قراءة من كاميرا الجهاز
- 💬 إشعارات واتساب فورية لولي الأمر

### 2️⃣ إدارة بيانات الطالب
- ➕ إضافة/تعديل/حذف الطلاب
- 🔢 كود تلقائي لكل طالب
- 📋 اختيار من قوائم المجموعات والصفوف
- 🖨️ طباعة الباركود

### 3️⃣ نظام الدفعات والفواتير
- 📅 تسجيل الدفعات حسب الشهر
- 📊 معرفة عدد الحصص المحضورة
- ✅ حالة الدفع (تم/لم يتم)
- 📈 تقارير الدفعات

### 4️⃣ نظام الدرجات
- 📝 إدخال درجات الطلاب
- 🎯 عدة امتحانات لكل طالب
- 📱 إرسال الدرجات عبر الواتساب
- 📊 تقارير الدرجات

### 5️⃣ التقارير
- 📋 تقارير الحضور والغياب
- 🎓 تقارير الدرجات
- 🖨️ طباعة التقارير

### 6️⃣ الإعدادات
- ⚙️ تخصيص اسم السنتر
- 💾 النسخ الاحتياطي والاسترجاع
- 🔄 إعادة ضبط المصنع

## المتطلبات
- Node.js 18+
- npm أو yarn

## التثبيت
```bash
npm install
```

## التشغيل
```bash
npm run dev
```
