# 🔧 دليل البناء والنشر الشامل\n\n## 📱 بناء APK (للهاتف الذكي)\n\n### ✅ الطريقة 1: Expo (الأفضل للمبتدئين)\n\n```bash\n# تثبيت Expo مرة واحدة\nnpm install -g expo-cli\n\n# بناء APK\nnpm run build:expo\n\n# أو استخدم أمر Expo مباشرة\nexpo build:android\n```\n\n**المميزات:**\n- ✅ بدون الحاجة لـ Android Studio\n- ✅ البناء يتم في السحابة\n- ✅ سريع جداً\n- ✅ سهل للغاية\n\n### ✅ الطريقة 2: البناء المحلي\n\n#### على Windows:\n```bash\nnpm run build:apk:windows\n```\n\n#### على Mac/Linux:\n```bash\nbash build-apk.sh\n```\n\n---\n\n## 🌐 بناء نسخة الويب\n\n```bash\n# بناء نسخة الويب\nnpm run build:web\n\n# النسخة الجاهزة ستكون في مجلد: dist/\n```\n\n### نشر الويب على:\n- **Vercel**: `vercel deploy dist/`\n- **Netlify**: اسحب وأسقط المجلد `dist/`\n- **GitHub Pages**: ارفع `dist/` على الفرع `gh-pages`\n\n---\n\n## 💻 بناء EXE (لسطح المكتب)\n\n### المتطلبات:\n```bash\nnpm install electron --save-dev\nnpm install electron-builder --save-dev\n```\n\n### البناء:\n```bash\nnpm run build:exe\n```\n\n---\n\n## 📋 متطلبات البناء:\n\n### للجميع:\n- ✅ Node.js 16+ ([تحميل](https://nodejs.org))\n- ✅ npm 8+ (يأتي مع Node.js)\n\n### لبناء APK محلياً:\n- ✅ Java Development Kit 11+\n- ✅ Android SDK\n- ✅ Android Studio (اختياري)\n\n### لبناء EXE:\n- ✅ Visual Studio Build Tools (لـ Windows)\n- ✅ Python 2.7\n\n---\n\n## 🚀 خطوات البناء السريعة:\n\n### 1. استنساخ المستودع:\n```bash\ngit clone https://github.com/mg9508861-max/ELHARAM.git\ncd ELHARAM\n```\n\n### 2. تثبيت المكتبات:\n```bash\nnpm install\n```\n\n### 3. البناء:\n```bash\n# لـ APK\necho \"🔨 جاري بناء APK...\"\nexpo build:android\n\n# أو لـ الويب\nnpm run build:web\n```\n\n### 4. النتيجة:\n- **APK**: سيكون رابط التحميل في الـ console\n- **Web**: الملفات في مجلد `dist/`\n- **EXE**: الملف في `release/`\n\n---\n\n## 📁 هيكل المشروع:\n\n```\nELHARAM/\n├── src/\n│   ├── components/      #
